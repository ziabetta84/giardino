# Progetti zona est

## Progetto insetti impollinatori

### Obiettivo

Trasformare la zona est in un'area a supporto degli insetti impollinatori (api, bombi, farfalle, sirfidi), con una fioritura scalare che copra il maggior numero di mesi possibile — dall'inverno fino all'autunno. La zona est, con la sua esposizione al sole mattutino e il microclima fresco e riparato, è adatta a specie che prediligono una luce non eccessivamente intensa.

### Piante coinvolte

**Già presenti e utili per gli impollinatori:**

| Pianta | Periodo fioritura | Impollinatori attratti |
|---|---|---|
| *Erica* spp. | inverno–primavera precoce | api, bombi |
| *Bellis perennis* | primavera–estate | api, sirfidi |
| *Primula* spp. | inverno–primavera | bombi, api |
| *Narcissus* spp. | primavera precoce | api, bombi |
| *Spiraea arguta* | primavera | api, farfalle |
| *Saxifraga* spp. | primavera | api, sirfidi |

**Da valutare per integrazione:**

Le seguenti specie sono state selezionate tenendo conto delle caratteristiche della zona est: sole mattutino (5–6 ore), ombra pomeridiana dalla collina, microclima riparato con umidità fogliare al mattino, escursione termica marcata e **suolo tendenzialmente asciutto** per la copertura della tettoia. Sono state escluse specie che richiedono suolo fresco o umido (es. *Helleborus*, *Pulmonaria*, *Digitalis*, *Geranium* selvatico).

- *Nepeta x faassenii* (o *N. racemosa*) — primavera–estate; api e bombi; ottima tolleranza alla siccità sistemica e alla mezza ombra, tra le poche specie che combinano queste due caratteristiche; fiori lilla-blu (varietà ornamentale, diversa dall'erba gatta *N. cataria*)
- *Centranthus ruber* — estate; farfalle e sfingi; si adatta benissimo a suoli asciutti e poveri, anche in posizioni parzialmente ombreggiate; fioritura lunga
- *Aster amellus* — autunno; api e farfalle; prolunga la fioritura fino a ottobre; tollera suoli ben drenati e asciutti, si adatta all'esposizione est

### Schema visivo

La zona è una **striscia stretta (~50 cm) che corre lungo tutta la parete est della casa**, con la tettoia che la copre parzialmente. Sul lato opposto si apre il prato. Le piante sono disposte linearmente.

```
[PARETE CASA / TETTOIA]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
 saxifraga │ narciso + primula │ erica │ spiraea
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
[PRATO APERTO]                    [COLLINA/RECINZIONE]
   ← sud                                    nord →
```

Le nuove piante da inserire vanno collocate nei tratti ancora liberi della striscia, rispettando le dimensioni contenute dello spazio disponibile.

### Stato attuale

In pianificazione. Le prime piante utili per gli impollinatori (erica, bellis perennis) sono già state introdotte. Il progetto non è ancora strutturato in modo sistematico.

### Prossimi passi

- Verificare i periodi esatti di fioritura delle piante già presenti durante la stagione
- Individuare i "vuoti" nella fioritura scalare (mesi scoperti: estate e autunno)
- Aggiungere una specie a fioritura estiva (*Nepeta* o *Centranthus ruber*) nei tratti liberi della striscia
- Aggiungere una specie a fioritura autunnale (*Aster amellus*) per prolungare la disponibilità di nettare
